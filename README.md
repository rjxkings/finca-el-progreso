# Finca Ganadera El Progreso

Sitio web de la finca ganadera El Progreso.
Hecho con HTML5, CSS3 y JavaScript (sin frameworks).

## Paginas

- `index.html` - Inicio
- `nosotros.html` - Historia, mision, vision, valores y equipo
- `servicios.html` - Servicios
- `productos.html` - Productos
- `galeria.html` - Galeria
- `contacto.html` - Contacto y formulario
- `informe.html` - Documento del proyecto (imprimir a PDF)

## Carpetas

- `css/` - estilos
- `js/` - scripts
- `images/` - logo

## Como verlo

Abrir `index.html` en el navegador.
