2. Introducción

La finca ganadera “El Progreso” es una empresa familiar dedicada a la producción de leche y carne bovina. Ante el aumento de su producción, la administración solicitó el desarrollo de un sitio web moderno, atractivo y responsive para dar a conocer sus servicios, productos y la calidad de sus procesos a clientes, inversionistas y visitantes.

Este proyecto se implementó utilizando únicamente HTML5, CSS3 y JavaScript, sin frameworks ni gestores de contenido, y se publicó de forma gratuita mediante GitHub Pages.

3. Objetivos

3.1 Objetivo general

Desarrollar un sitio web estático profesional para la finca ganadera “El Progreso”, aplicando buenas prácticas de diseño web, organización del código y publicación del proyecto mediante GitHub Pages.

3.2 Objetivos específicos

  • Diseñar una identidad visual acorde a la temática ganadera.
  • Implementar las secciones solicitadas: Inicio, Nosotros, Servicios, Productos, Galería y Contacto.
  • Garantizar usabilidad en dispositivos móviles y de escritorio.
  • Organizar el código fuente de forma clara y mantenible.
  • Publicar el sitio en GitHub Pages y documentar el proceso.

7. Conclusiones

Con este proyecto se logro crear un sitio web funcional para la finca El Progreso usando HTML5, CSS3 y JavaScript, cumpliendo con las secciones pedidas por el cliente y un diseno adaptable a celular y computadora.

El codigo quedo organizado en archivos separados y el sitio se publico con GitHub Pages para que cualquier persona pueda verlo desde Internet.

Como mejora se podria conectar el formulario de contacto a un servicio real de correo y agregar mas fotos propias de la finca.
